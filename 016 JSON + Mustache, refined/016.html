016: JSON + Mustache, Refined

After getting some help from a JS ninja at work, this project (016) simply cleans up most of the JSON/AJAX/JS/Mustache flow I'd fallen into using the past 5-6 projects. Templates are now stored in the HTML, and the data source (.json file) is loaded in a new way that ensures it won't ever fall victim to unintended async issues, which my old code had.

Otherwise this project recreates the function of 012 by displaying this JSON data in HTML via Mustache template.
