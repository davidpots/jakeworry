InView Plugin

Super simple and boring: all this page does is look for item `#target1` and gives it the class `visible` when it enters the viewport (with a threshold of 100px, so you can actually see the class name being added via CSS background color). For more info on the plugin, see: http://henkbulder.github.io/inViewPort/
