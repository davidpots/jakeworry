017: JSON Grouping, part 3

In which I take the work from 014 + 015 and add some more refinement and complexity. Again, this starts with a single JSON data file and manipulates the data to group songs, this time by multiple criteria (title, artist, and year). I also used replace/regex for the first time in JS -- this handles both titles/artists that begin with "The " (to ignore that word) as well as grouping all songs starting with any number into the "#" group. Finally, I have also been given some notes on how all of this can be drastically cleaned up -- which I plan on implementing in 018.
