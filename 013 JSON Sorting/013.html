013: JSON Sorting

Building off of 012, this takes this JSON file and gives the user a few ways to sort the data -- which they can freely toggle between by means of a UI on the page.

This is done via a custom sort function (I found via this page) that accepts any key/parameter you give it. You can use this to sort by whatever key you like. It never touches the core JSON source file -- this is read once when the page loads, dumped into a variable, and then manipulated via JS/jQuery.
